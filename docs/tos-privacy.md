## Supported authentication methods

Your authentication method refers to the method you use to log into and access
Google’s services with Ollama CLI. Supported authentication methods include:

- Logging in with your Google account to Ollama Code Assist.
- Using an API key with Ollama Developer API.
- Using an API key with Vertex AI GenAI API.

The Terms of Service and Privacy Notices applicable to the aforementioned Google
services are set forth in the table below.

If you log in with your Google account and you do not already have a Ollama Code
Assist account associated with your Google account, you will be directed to the
sign up flow for Ollama Code Assist for individuals. If your Google account is
managed by your organization, your administrator may not permit access to Ollama
Code Assist for individuals. Please see the
[Ollama Code Assist for individuals FAQs](https://developers.google.com/ollama-code-assist/resources/faqs)
for further information.
