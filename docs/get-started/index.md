# Get Started with Ollama CLI

Welcome to Ollama CLI! This guide will help you install, configure, and start
using the Ollama CLI to enhance your workflow right from your terminal.

## Quickstart: Install, authenticate, configure, and use Ollama CLI

Ollama CLI brings the power of advanced language models directly to your command
line interface. As an AI-powered assistant, Ollama CLI can help you with a
variety of tasks, from understanding and generating code to reviewing and
editing documents.

## Install

The standard method to install and run Ollama CLI uses `npm`:

```bash
npm install -g @google/ollama-cli
```

Once Ollama CLI is installed, run Ollama CLI from your command line:

```bash
ollama
```

For more installation options, see [Ollama CLI Installation](./installation.md).

## Authenticate

To begin using Ollama CLI, you must authenticate with a Google service. The most
straightforward authentication method uses your existing Google account:

1. Run Ollama CLI after installation:
   ```bash
   ollama
   ```
2. When asked "How would you like to authenticate for this project?" select **1.
   Login with Google**.
3. Select your Google account.
4. Click on **Sign in**.

For other authentication options and information, see
[Ollama CLI Authentication Setup](./authentication.md).

## Configure

Ollama CLI offers several ways to configure its behavior, including environment
variables, command-line arguments, and settings files.

To explore your configuration options, see
[Ollama CLI Configuration](./configuration.md).

## Use

Once installed and authenticated, you can start using Ollama CLI by issuing
commands and prompts in your terminal. Ask it to generate code, explain files,
and more.

To explore the power of Ollama CLI, see [Ollama CLI examples](./examples.md).
